# PROJECT_BRIEF.md

## UangKu: Project Brief

### Solving Indonesia's "Scattered Money" Problem

---

### Executive Summary

**UangKu** is a mobile application that transforms how Indonesians manage their finances. Instead of manually checking balances across 5+ financial platforms (banks + e-wallets), UangKu automatically consolidates all transactions into a single, real-time dashboard through intelligent notification interception.

### Problem: The "Scattered Money" Crisis

#### Current Pain Points

Indonesian users face a fragmented financial ecosystem:

1. **Multiple Accounts**:
   - 3–7 bank accounts across BCA, Mandiri, BRI
   - 4–6 e-wallet accounts: GoPay, OVO, Dana, Jago, LinkAja, Shopee
   - Result: **No unified view of total money**

2. **Manual Tracking**:
   - Users must open each app individually to check balances
   - Transaction history is spread across different interfaces
   - High friction and error-prone record-keeping

3. **Financial Anxiety**:
   - Unclear how much money they actually have
   - Difficult to identify spending patterns
   - Hard to budget effectively across platforms

4. **Time Wasted**:
   - Average user spends 15–20 minutes daily checking balances
   - Mental fatigue from context-switching across apps

#### Market Opportunity

- **Total Market**: 100+ million active Indonesian digital users
- **Target Segment**: 25 million users with 3+ financial accounts
- **Monetization Ready**: Future features include analytics, bill reminders, investment tracking

### Solution: UangKu's Unique Value Proposition

#### Core Innovation: Notification Listening

Instead of **asking** users to upload transactions, UangKu **listens** to transaction notifications from banks and e-wallets. This is:

- **Automatic**: No manual entry needed
- **Real-time**: Transactions appear instantly
- **Secure**: We never store payment credentials

#### User Experience Highlights

1. **One-Tap Setup**:
   - Google Sign-In (no new password)
   - Select your wallets from a pre-populated list
   - Done in under 2 minutes

2. **Minimalist Dashboard**:
   - Single view of total balance
   - Quick summary: Income / Expense / Net balance
   - Transaction list with date navigation

3. **Smart Confirmation**:
   - Each transaction sends a push notification for approval
   - Prevents duplicates and parsing errors
   - Builds user trust

4. **One-Hand Navigation**:
   - Bottom tab bar for easy thumb access
   - Large touch targets (Material Design 3)
   - Dark mode for reduced eye strain

#### Key Features (MVP)

| Feature                           | Benefit                                                       |
| --------------------------------- | ------------------------------------------------------------- |
| **Real-Time Balance Aggregation** | Users see total money across all accounts instantly           |
| **Transaction History**           | Complete audit trail of all recorded transactions             |
| **Wallet Management**             | Create, edit, delete wallets for different financial accounts |
| **Monthly Reports**               | Visual breakdown of income vs. expense with trend analysis    |
| **Notification Settings**         | User controls which apps to listen to                         |
| **Google Authentication**         | Secure login without password fatigue                         |
| **Dark Mode**                     | Comfortable to use anytime, anywhere                          |

### Why UangKu Will Win

#### 1. **Market Fit**

- Indonesian users are digitally savvy and use multiple financial platforms
- Manual consolidation is a real, daily pain point
- No strong competitor addressing this specific use case

#### 2. **Technical Advantage**

- Notification interception is hard to replicate (requires deep OS-level Android integration)
- Android-only platform ensures focused, optimized development
- Expo + React Native allows rapid iteration and testing
- Headless services ensure reliability even when app is closed

#### 3. **Trust & Security**

- Never stores payment credentials
- Google authentication provides instant credibility
- Transparent notification confirmation prevents user distrust

#### 4. **Indonesian-First Approach**

- UI designed for Indonesian users (Material Design, accessibility for lower-end devices)
- In-app text is friendly and relatable, not corporate jargon
- Supports Indonesian number formatting and currency

#### 5. **Network Effects**

- Referral program incentivizes friends to join
- Community insights (e.g., "average Indonesian saves X per month") build engagement
- Integration with financial advisors creates B2B opportunity

### Conclusion

UangKu is not just a financial app—it's a **financial liberator** for Indonesians. By automatically consolidating scattered funds, UangKu gives users clarity, confidence, and control over their money.

**Vision**: To become the default financial dashboard for every Indonesian with a smartphone.
